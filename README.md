# Caliche Pan - Prototipo HTML, CSS y JavaScript

Prototipo web tipo app móvil para una panadería artesanal. Incluye pantalla de bienvenida, inicio de sesión, registro, catálogo, carrito, historial y perfil.

## Estructura
- `index.html`: página principal.
- `login.html` y `registro.html`: autenticación simulada.
- `catalogo.html`: listado de productos y buscador.
- `carrito.html`: productos agregados, cantidades y total.
- `perfil.html`: datos del usuario.
- `css/styles.css`: estilos, colores, diseño responsive y componentes.
- `js/app.js`: interacción, búsqueda, carrito y localStorage.
- `assets/`: imágenes utilizadas por el prototipo.

## Ejecución local
Abrir `index.html` en el navegador o usar la extensión Live Server de Visual Studio Code.

## Publicación sugerida
Subir la carpeta a un repositorio de GitHub y activar GitHub Pages desde Settings > Pages > Deploy from branch > main.
